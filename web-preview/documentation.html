üîß Troubleshooting

Common Issues

Button not responding on Android 12+

Problem: Button feels unresponsive or gets stuck
Solution: Ensure Android 12+ optimizations are working and error recovery is enabled.
<UniversalButton title="Android 12+ Button" onPress={handlePress} enableErrorRecovery={true} // ‚úÖ Enable automatic recovery />

Icons not displaying

Problem: Icons not showing up
Solution: Check Material Icons installation and icon name spelling.
// ‚úÖ Verify Material Icons are installed <UniversalButton title="Icon Button" iconName="star" // ‚úÖ Check spelling iconSize={24} // ‚úÖ Ensure reasonable size iconColor="#FFD700" // ‚úÖ Ensure visible color />

Performance issues

Problem: Button feels slow or janky
Solution: Enable performance optimizations and use memoization.
// ‚úÖ Enable performance tracking <UniversalButton title="Performance Button" onPress={handlePress} trackPerformance={true} enablePerformanceHandling={true} /> // ‚úÖ Use memoization for frequently rendered buttons const MemoizedButton = React.memo(({ title, onPress }) => ( <UniversalButton title={title} onPress={onPress} /> ));

Styling not applied

Problem: Custom styles not working
Solution: Use correct style props (style and textStyle objects).
// ‚ùå Wrong: Individual style props <UniversalButton width="100%" // ‚ùå Not supported fontSize={18} // ‚ùå Not supported /> // ‚úÖ Correct: Use style and textStyle objects <UniversalButton style={{ width: '100%' }} // ‚úÖ Container styling textStyle={{ fontSize: 18 }} // ‚úÖ Text styling />
